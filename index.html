General features

Here, in WPTheme4.me, we took minimalist design principles to keep things simple and focus only for elements that are absolutely essential to communicate a message to a website visitor - your visual content. Content comes first in minimalist design.

Clean, beautiful and minimalist Wordpress themes perfectly suits for whose who put their work in front.

Minimalist approach is applied not only to layout a visual content in frontend, but for a content management in backend too. Creating, managing your online portfolio is easier than ever and it is done in native Wordpress environment. No need to learn how to customize dozens of theme settings. We bring just enough functionality: add text and visual content (iliustrations, architectural drawings, videos, photos). Theme will take care of the rest.

Responsive layout is de-facto in today's website design. Your visual content, be it video, picture or photo, will adjusts to any screen size. Did we mentioned that it works on all (last 3 versions) browsers? Your Wordpress site will look the same on Internet Explorer, Chrome, Firefox and Safari browsers.

Easy and minimal setup: activate theme, add your logo, favicon and contact information. That's it.

If you wonder how do we get theme names here is the story. Our plan is to craft many, no, dozen Wordpress themes. Needed a system how to name them. In the beggining there were idea just use numbers. But.. Wordpress creators already doing this (default themes are named by the year) and it perhaps will arouse misunderstandings for our support team and clients.

What about minimalism? Minimalist artist names? Wordpress release codenames are after well-know jazz musicians. We started to read about each artist. It took a while.. too long. We started to read more and do less. We had to change the direction. What about the shapes? Colors? We did some search in Wordpress theme repository to check how many themes do have color names. Found some, but not many. Plenty names to choose from. Finally!!!
